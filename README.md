# Gesture PID Control System

双模态手势识别与PID云台控制系统：Python端实时手势识别 → 串口通信 → STM32固件PID控制。

## 系统架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   摄像头    │───▶│ MediaPipe   │───▶│ 自适应EMA   │───▶│  线性预测   │
│  (手势输入) │    │ 手部21点检测 │    │  +离群点滤波│    │  (前馈控制) │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                                │
                                                                ▼
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   云台/机械臂│◀───│  TIM3 PWM    │◀───│   PID控制    │◀───│  串口接收    │◀───
│   (执行机构) │    │  (50Hz输出)  │    │  (增量式PID) │    │ (#Xcoords$Y)│
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

## 核心创新点

### 1. 自适应EMA滤波
- 根据运动速度动态调整滤波系数
- 慢速运动：α=0.05（深度平滑）
- 快速运动：α=0.15（快速响应）

### 2. 二阶离群点检测
- 一阶检测：单帧坐标突变 >50像素时降α
- 二阶检测：速度突变 >历史3倍时极度保守

### 3. 线性预测控制
- 基于速度外推预测下一帧位置
- 预测偏差 >80像素时自动回退到平滑值
- 速度 <2像素/帧时不预测（防噪声放大）
- **理论关联**：前馈控制 + Smith预估器思想

## 文件结构

```
.
├── control/                    # STM32F103ZE 固件 (Keil MDK)
│   ├── USER/main.c            # 主程序 + PID控制环
│   ├── HARDWARE/PID/          # PID控制实现
│   ├── HARDWARE/serial/       # 串口接收 (#X$Y\r\n)
│   └── HARDWARE/TIMER/        # TIM3 PWM输出 (50Hz)
│
└── svm/                        # Python手势识别
    ├── HandTrackingModule.py   # 核心模块（检测+滤波+预测）
    ├── inference.py            # 实时推理入口
    ├── collect_data.py         # 数据采集
    ├── train_model.py          # SVM模型训练
    └── dataset.csv             # 训练数据集
```

## 快速开始

### Python端

```bash
cd svm
pip install opencv-python mediapipe scikit-learn joblib numpy

# 采集新手势数据
python collect_data.py

# 训练模型
python train_model.py

# 运行实时推理
python inference.py
```

### STM32固件

1. 用Keil MDK打开 `control/USER/CONTROL.uvprojx`
2. Build目标：**USART**
3. 通过ST-Link/J-Link烧录到STM32F103ZE

## 串口协议

| 参数 | 值 |
|------|-----|
| 波特率 | 115200 |
| 格式 | `#X坐标$Y坐标\r\n` |
| PWM频率 | 50Hz (ARR=9999, PSC=143) |

## PID参数

| 参数 | X轴 | Y轴 |
|------|-----|-----|
| Kp | 0.04 | 0.05 |
| Ki | 0 | 0 |
| Kd | 0.30 | 0.30 |
| PWM范围 | [300, 1200] | [300, 1000] |
